ABOUT

A very simple Python/Html/CSS recital program generator.
 ! IMPORTANT !
you must create a .csv file with a very specific format for it to work
Row 1
    col 1: should have the word "Headers" written
    col 2: studio name
    col 3: title
    col 4: date
    col 5: subtitle 1
    col 6: footer

All subsequent rows should have:
    col 1: performers
    col 2: titles
    col 3: composers

For multiple pieces separate the titles AND composers using the "+" sign
For multiple movements separate using the "@" sign
